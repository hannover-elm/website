Hannover Elm Language Meetup

This is a meetup for anyone interested in the Elm programming language (https://elm-lang.org).

We are organizing a monthly meetup where we can share with and learn from each other. We will be getting together to discuss our Elm projects, help each other with problems and if you are a newcomer we will be more than happy to provide guidance on learning the language.

Elm is a delightful language for reliable web applications. It comes with features similar to those of popular JavaScript frameworks like React, Vue.js or Angular, and it is strongly-typed, giving you similar benefits like those of TypeScript or Flow.

Elm Language Meetup: Online Beginner's Night 2021-07-21 19:00

ONLINE EVENT. Please join us at https://meet.foremny.me/HannoverElmLanguageMeetup .

If you want to learn the Elm programming language, then this event is ideal for you. We usually start with a small introduction to the programming language itself, and finish with a courageous volunteer to program their first Elm program.

You don't have to bring any prior experience in web development or programming in general. It would be good if you brought a laptop to program on though.

If you are a beginner that does not immediately want to jump into programming, that is OK as well. Depending on the participants we are flexible to adjust our program.

Agenda:
- An Introduction to the Elm Programming Language
- Guide: Setup of a Development Environment
- Crowd-Coding: Hello World
